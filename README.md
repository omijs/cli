## omi-cli

### New Project base on TypeScript

```bash
$ npx omi-cli init my-app
$ cd my-app           
$ npm start           # develop
$ npm run build       # release
```

### New Project base on TypeScript + Tailwindcss

```bash
$ npx omi-cli init-tw my-app
$ cd my-app           
$ npm start           # develop
$ npm run build       # release
```


### New Project base on JavaScript

```bash
$ npx omi-cli init-js my-app
$ cd my-app           
$ npm start           # develop
$ npm run build       # release
```
